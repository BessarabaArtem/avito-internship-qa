# Тест-кейсы API (Microservice)

В этом документе описаны проверки для микросервиса объявлений.
**Host:** `https://qa-internship.avito.com`

## 1. POST /item (Создание объявления)

| ID | Описание проверки | Body (JSON) | Ожидаемый результат |
|:---|:---|:---|:---|
| **1.1** | **Успешное создание (Valid)** | `{ "sellerID": 112233, "name": "Phone", "price": 999 }` | **200 OK**. <br>Ответ содержит `status` и уникальный `id`. |
| **1.2** | **Граничное значение SellerID (Min)** | `{ "sellerID": 111111, "name": "Test", "price": 100 }` | **200 OK**. Объявление создано. |
| **1.3** | **Граничное значение SellerID (Max)** | `{ "sellerID": 999999, "name": "Test", "price": 100 }` | **200 OK**. Объявление создано. |
| **1.4** | **Негативный: Цена меньше нуля** | `{ "sellerID": 112233, "name": "Test", "price": -50 }` | **400 Bad Request**. Ошибка валидации цены. |
| **1.5** | **Негативный: Пустое название** | `{ "sellerID": 112233, "name": "", "price": 100 }` | **400 Bad Request**. Ошибка валидации названия. |
| **1.6** | **Негативный: Некорректный тип (String in Price)** | `{ "sellerID": 112233, "name": "Test", "price": "free" }` | **400 Bad Request**. Ошибка типа данных. |

## 2. GET /item/{id} (Получение объявления)

| ID | Описание проверки | Параметры | Ожидаемый результат |
|:---|:---|:---|:---|
| **2.1** | **Получение существующего объявления** | `id` = *существующий ID* | **200 OK**. <br>JSON ответа совпадает с созданными данными. |
| **2.2** | **Запрос несуществующего ID** | `id` = `00000000-0000-0000-0000-000000000000` | **404 Not Found**. |
| **2.3** | **Невалидный формат ID** | `id` = `invalid-id-string` | **400 Bad Request**. |

## 3. GET /{sellerID}/item (Объявления продавца)

| ID | Описание проверки | Параметры | Ожидаемый результат |
|:---|:---|:---|:---|
| **3.1** | **Получение списка объявлений** | `sellerID` = *существующий продавец* | **200 OK**. <br>В массиве ответа есть созданное объявление. |
| **3.2** | **Продавец без объявлений** | `sellerID` = `999000` | **200 OK**. <br>Пустой массив `[]`. |
| **3.3** | **Некорректный SellerID (Буквы)** | `sellerID` = `abc` | **400 Bad Request**. |

## 4. GET /statistic/{id} (Статистика)

| ID | Описание проверки | Параметры | Ожидаемый результат |
|:---|:---|:---|:---|
| **4.1** | **Получение статистики** | `id` = *существующий ID* | **200 OK**. <br>В ответе поля: `likes`, `viewCount`, `contacts`. |
| **4.2** | **Статистика несуществующего ID** | `id` = `00000000-0000-0000-0000-000000000000` | **404 Not Found**. |
